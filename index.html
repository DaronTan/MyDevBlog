"border" css styling. This can create a border around a paragraph.

"id" and "class" selectors. provide tags with selectors to be selected and styled in css.


"@media" Media queries, can be used to modify the CSS of a website depending on viewport size/device type.


"position" positionings. There are 4 types of positionings; static, relative, absolute and fixed.


"z-index" z-index can control depth of elements on website; can create navigation bars on websites that stay onscreen for users.


z-index has positionings.

  • Relative
  • Absolute
  • Relatively Absolute
  • Fixed

"flex" flexbox. Flexbox containers are parents of flex items.


Wireframe. wireframes are good for planning out a website format & appearance in advance (wireframe is not design)


Wireframes can be created static with apps such as Adobe XD or drawn out. They are not responsive but can be quick to make.


Wireframes have 3 types of fidelity: Low to High.
