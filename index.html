Paragraph tag "p" can create a paragraph of sentence(s)

Link tag "a" can be used to put hyperlinks on the webpage.

  • is

Ordered list tag "ol" can be used to create a list with order.

Sematic-neutral elements -> "div" & "span" provides structure to the page.

This paragraph is encapsulated by a div element right now. You probably can't tell but you just have to take my word for it :)

Table tag "table" can be used to create a table.

Tag Purpose
"tr" table row
"th" table header
"td" table data

inline frame tag "iframe" can be used to put a video on the webpage.


CSS

Linking external style sheet "Link" (.CSS files)


Colors property "color" can be used in paragraphs, headers etc to change the color of texts.

This sentence is light blue.


Font family "font-family" can be used to change the font of headers, paragraphs etc.

This sentence is in Times New Roman.


id / class attributes "id=' ' / class=' '" can be used to target individual elements to modify them.

This sentence has the id 'potato'. You probably can't tell but the id is indeed 'potato'.


Background color property "background-color" can be used to change the background color of paragraphs, headers, background etc.

This sentence has the background color 'lightgoldenrodyellow.'
